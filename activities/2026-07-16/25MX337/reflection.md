# Reflection

## Mission Tata 1 Reflection

This mission helped me understand how developers use Git to manage projects and collaborate with others. Before this activity, I knew Git only at a basic level, but now I understand the complete workflow of creating files, tracking changes, committing updates, and pushing them to a remote repository.

One important learning was that documentation is just as important as writing code. A well-written README helps anyone understand the project without needing additional explanation.

I also learned that commits act as checkpoints, making it easier to track progress and revert changes if needed.

During this mission, I practiced organizing project files, writing documentation using Markdown, and following a structured repository layout.

Overall, this mission improved my confidence in using Git and GitHub. These skills will be valuable for future academic projects, internships, and software development roles where version control and collaboration are essential.
